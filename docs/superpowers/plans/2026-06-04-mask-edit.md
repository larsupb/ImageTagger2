# Mask Edit Mode Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a mask edit mode to the Edit page that lets users paint white/black strokes onto the mask overlay canvas and save the result as the new mask file.

**Architecture:** Extends the existing Paint mode pattern тАФ a canvas overlay positioned over the rendered image area, mouse event handlers in `ImageViewer`, state and save logic in `page.tsx`. Key differences from paint: canvas is pre-loaded with the existing mask, the eraser paints black (not transparent), and save replaces the mask file rather than compositing onto the image.

**Tech Stack:** Python/FastAPI/Pillow (backend), Next.js 15/React 19/TypeScript/Tailwind/shadcn (frontend), HTML5 Canvas 2D API.

```python
@router.post("/mask/save")
async def save_mask_endpoint(
    index: int = Form(...),
    mask_png: UploadFile = File(...),
    session: Session = Depends(get_session),
):
    if session.dataset is None:
        raise HTTPException(400, "No dataset loaded")
    try:
        data = await mask_png.read()
        processing_service.save_mask(session, index, data)
        return {"status": "saved", "index": index}
    except Exception as e:
        logger.error("mask save failed for index %s: %s", index, e, exc_info=True)
        raise HTTPException(500, str(e))
```

Add this helper after the existing `stamp` function definition:

```typescript
  function stampMask(canvas: HTMLCanvasElement, x: number, y: number) {
    const ctx = canvas.getContext("2d");
    if (!ctx) return;
    const tool = maskEditTool ?? "pencil";
    const size = maskEditSize ?? 8;
    const half = size / 2;
    ctx.globalCompositeOperation = "source-over";
    ctx.fillStyle = tool === "eraser" ? "black" : "white";
    if (tool === "quad") {
      ctx.fillRect(x - half, y - half, size, size);
    } else {
      ctx.beginPath();
      ctx.arc(x, y, half, 0, Math.PI * 2);
      ctx.fill();
    }
  }
```

```typescript
    const canvas = maskCanvasRef.current;
    if (!canvas) return;
    setProcessing("mask_save");
    try {
      const blob = await new Promise<Blob>((resolve, reject) => {
        canvas.toBlob(
          (b) => (b ? resolve(b) : reject(new Error("Canvas empty"))),

      setMaskEditMode(false);
      setMaskVersion((v) => v + 1);
      loadItem(safeIndex);
      queryClient.invalidateQueries({ queryKey: ["versions", safeIndex] });

    } finally {
```
